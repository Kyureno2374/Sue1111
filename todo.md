🗄️ BACKEND (SUPABASE)
📌 1. Таблицы
users — аутентификация и пользовательские данные
Содержит следующую информацию:

id: уникальный идентификатор (UUID)

username: имя пользователя

password: хэшированный пароль

wallet_address: адрес криптокошелька

balance: текущий баланс пользователя

games_played: количество сыгранных игр

games_won: количество выигранных игр

deposits: сумма всех депозитов

verified: статус верификации (true/false)

created_at: дата и время регистрации

games — информация об играх
Содержит следующую информацию:

game_id: уникальный ID игры (UUID)

creator_user_id: ID создателя игры (ссылка на users)

players: список участников (JSONB)

status: статус игры (active, completed, cancelled)

created_at: дата создания

updated_at: дата последнего обновления

winner_user_id: ID победителя (если завершена)

game_params: параметры игры (JSONB)

leaderboard — агрегированные данные (можно пересчитывать на сервере или cron)
Содержит следующую информацию:

user_id: ID пользователя (связь с users)

total_wins: общее число побед

total_earnings: общий выигрыш в валюте

win_rate: процент побед

transactions — финансы и история
Содержит следующую информацию:

transaction_id: уникальный ID транзакции

user_id: ID пользователя

type: тип транзакции (deposit, withdrawal, game_entry, reward)

amount: сумма транзакции

status: статус (pending, approved, rejected)

created_at: дата и время создания

withdraw_requests — управление выводами
Содержит следующую информацию:

request_id: уникальный ID запроса

user_id: ID пользователя

amount: сумма к выводу

wallet_address: адрес кошелька для вывода

status: статус запроса (pending, processed, rejected)

created_at: дата и время создания

notifications — уведомления
Содержит следующую информацию:

id: уникальный ID уведомления

user_id: ID пользователя, которому отправлено уведомление

type: тип уведомления (например: system, withdrawal, game_result)

message: текст уведомления

read: статус прочтения (true/false)

timestamp: дата и время создания уведомления
